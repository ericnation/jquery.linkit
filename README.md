# LinkIt jQuery Plugin  

LinkIt is a simple jQuery plugin to attach links to elements.

## Version 
0.1.0

##Usage 
    <script>
		$(document).ready(function() {
			$('element').linkIt({
				href: 'http://test.com',
				text: 'Click me',
				target: '_blank'
			})
		});
	</script>
